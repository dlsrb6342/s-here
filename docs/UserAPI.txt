User API
    •   회원가입 ( /api/user/signup POST )
    content type : application/json
        {
            studentId :  ,
            password :  ,
            name :  ,
            email :  
        }

    return value :
        요청에 성공했을 때 : { success: true }
        실패했을 때 : {error : , code : }
            code == 0 : 스마트카 트랙 이수 학생이 아닐 때
            code == 1 : 이미 학번이 존재할 때
            code == 2 : 비밀번호가 형식에 맞지 않을 때
            code == 3 : 등록되어있는 이름과 같지 않을 때


    •   로그인 ( /api/user/login POST )
    content type : application/json
        {
            studentId :  ,
            password :  
        }

    return value : 
        요청에 성공했을 때 : { success: true, confirmed: }
            이메일 인증 여부에 따라 confirmed 값(true, false)이 정해진다.
        실패했을 때 : {error : , code : }
            code == 0 : 비밀번호가 틀렸거나 학번이 틀렸을 때
            code == 1 : 스마트카 트랙 이수 학생이지만 회원가입을 안했을 때


    •   이메일 인증 ( /api/user/confirm POST )
    content type : application/json
        {
            token :   
        }

    return value : 
        요청에 성공했을 때 : { success: true }
        실패했을 때 : {error : , code : }
            code == 0 : 잘못된 토큰이 왔을 때


    •   로그아웃 ( /api/user/logout POST)
    return value : 
        요청에 성공했을 때 : { success: true }
        실패했을 때 : {error : , code : }
            code == 0 : 로그인되어있지 않을 때
