User API
    •   회원가입 ( /api/user/signup POST )
    content type : application/json
        {
            studentId :  ,
            password :  ,
            name :  ,
            email :  
        }

    return value :
        요청에 성공했을 때 : { success: true }
        실패했을 때 : {error : , code : }
            code == 0 : 이미 학번이 존재할 때
            code == 1 : 비밀번호가 형식에 맞지 않을 때


    •   로그인 ( /api/user/login POST )
    content type : application/json
        {
            studentId :  ,

        }
